# Nmap Port Scanner GUI

## 📌 Description
This project is a GUI-based port scanner built using Python. It helps users scan open ports of a target system using Nmap.

## 🎓 Internship Project
This project is developed as part of the Vodafone Idea (VOIS) AICTE Internship Program.

## 🚀 Features
- Simple GUI interface
- Scan open ports
- Fast and easy to use

## 🛠️ Technologies Used
- Python
- Nmap
- Tkinter (GUI)

## ▶️ How to Run
1. Install Nmap
2. Install Python
3. Run the Python file:
   python main.py
